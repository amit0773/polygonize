# Polygonize

Any text convert to svg polygon coordinates

## Installation

Add this line to your application's Gemfile:

    gem 'polygonize'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install polygonize

## Usage

    include Polygonize
    ORIGINAL_TEXT.coordinates
    # => "x1,y1 x2,y2 x3,y3....."
